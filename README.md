# Vue Study Project

Projeto desenvolvido para praticar e consolidar conhecimentos em Vue.js 3 através de exercícios práticos.

A ideia deste repositório é começar com aplicações simples e evoluir gradualmente o projeto conforme novos conceitos da tecnologia forem aprendidos e aplicados.

---

## 📚 Objetivo

Este projeto faz parte do meu processo de aprendizado em Vue.js 3.

Cada funcionalidade implementada tem como objetivo colocar em prática conceitos importantes da framework, como:

- Reatividade
- Computed Properties
- Diretivas
- Renderização dinâmica
- Manipulação de estado

---

## 🛒 Primeiro exercício: Carrinho de compras

O primeiro exercício do projeto consiste em um pequeno sistema de carrinho de compras onde é possível:

- visualizar produtos
- definir a quantidade desejada
- calcular automaticamente o valor total

Este exercício foi criado para praticar principalmente **computed properties** e **reatividade** no Vue.
